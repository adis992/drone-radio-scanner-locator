🎮 HOW TO USE
═══════════════════════════════════════════════════════════════════

1. Start Application:
   ./start.sh

2. Open Browser:
   http://localhost:3001

3. Check RTL-SDR Status:
   Top right corner shows connection status

4. Start Scanning:
   Click "Start Full Scan" button

5. View Devices:
   Devices appear as cards in real-time
   - Drone detected? You'll see it!
   - IoT sensor nearby? It appears!
   - Bluetooth device? Listed!

6. Listen to Signals:
   Click speaker icon on any device card
   Hear the actual transmission live!

7. Record Signals:
   Click record button on any device
   Saves to recordings/ folder

8. Manage Recordings:
   Click "Recordings" button (top right)
   Download or replay any recording

9. Filter Devices:
   Use dropdown to filter by type
   (All, Drones, IoT, Bluetooth, etc.)

10. Stop Scanning:
    Click "Stop Scan" when done

💡 PRO TIPS
═══════════════════════════════════════════════════════════════════

• Use external antenna for better range
• USB 2.0 ports more stable than 3.0
• Lower frequencies penetrate buildings better
• ADS-B works best outdoors
• Bluetooth detection works best close range
• Check recordings folder for saved audio
• Try different times of day for different signals
• Drones typically use 2.4 GHz and 5.8 GHz
